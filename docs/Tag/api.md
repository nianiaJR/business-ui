### 接口描述 - Tag
| 属性          | 说明                                                               | 类型                    | 必须 | 默认值 |
| ------------- | ---------------------------------------------------------------- | ----------------------- | ---- | ------ |
| onChange | tags值发生变化事件触发 | (value: any) => void | N    | -      |
| addText | 添加按钮文案 | string | N    | -      |
| disabled | 是否处于禁止状态 | boolean      | N    | -      |
| value | 是否控制tags的值 | Array<string> | N    | - |
| defaultValue | 默认值 | Array<string> | N    | -      |
